注意事项

新闻动态 admin
指数
80

1、点击标题即可查看该条新闻；
2、点击操作栏即可完成操作(发布：即可发布新闻到微信公众号。取消：即可取消已经发布了的新闻)；
3、发布状态与待审核状态颜色不同；
4、可以输入关键字精确的查找。关键字会在新闻内容和标题高亮；
5、可以在查看单条新闻点击下方的：查看原文 查看来源的该条新闻。

种子网站 admin
指数
60

1、可以指定爬取的网站；
2、状态栏有小勾即为选中该种子网站；
3、选中网站点击更新即可完成对爬取网站的指定。
